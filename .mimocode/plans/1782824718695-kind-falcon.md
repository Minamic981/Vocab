# Export Words Feature

## Goal
Add a button to export the word list as a plain text file in `english = persian` format (one word per line).

## Changes

### 1. HTML — `templates/components/body.html`
Add an export button in the Library tab, next to the search bar area.

### 2. JS — `static/app.js`
Add click handler that:
- Builds a string from the `words` array: `"english = persian\n"` per line
- Creates a Blob with `text/plain` type
- Triggers a download via a temporary `<a>` element with `URL.createObjectURL`
- Filename: `vocabulary.txt`

### 3. CSS — `static/style.css`
Style the export button (small ghost button, matching existing `.btn-ghost .btn-sm` pattern).

## Files to modify
- `templates/components/body.html` (add button)
- `static/app.js` (add handler)
- `static/style.css` (minimal, if needed)

## Verification
- Open the app in browser
- Add a few words
- Click Export → downloads `vocabulary.txt`
- Open the file → verify format is `english = persian`, one per line
